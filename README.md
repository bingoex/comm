# Comm Lib #

一个简单C语言库<br/>
A common lib in C language for all C\* project 


## cm_cfg
	a config file parse module

## cm_log
	a log module

## cm_shm
	a system V shm tool module

## cm\_debug
	提供以16进制打印sBuf的API

## cm_net
	a net sock module

## cm_process
	a process module

## cm_math
	a math module

## cm\_binary
	提供简单易用的二进制封包解包API
	
## cm\_time
	提供tm转字符串API

## cm\_type
	提供统一的类型宏定义
	INT32

## cm_procenum_ctol
	control the number of proceess in case of too many process be start

## cm_lock
	a lock module that use semaphore
